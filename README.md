# 🚀 Smart To-Do App (Streamlit)

A simple and interactive **To-Do List web app built with Python and Streamlit**.
It helps you organize tasks, track progress, and stay productive.

## ✨ Features

* 🧭 Sidebar navigation
* ➕ Add new tasks
* ✔ Mark tasks as completed
* 🗑 Delete tasks
* ⚡ Priority levels
* 📊 Progress tracking with progress bar
* 🎈 Celebration animation on task completion

## 🛠 Built With

* **Python**
* **Streamlit**

## ▶️ Run the App

```bash
streamlit run app.py
```

This project demonstrates how to build a **simple productivity app using Streamlit** with session state and interactive UI components.
